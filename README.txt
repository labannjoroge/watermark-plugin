A comprehensive WordPress plugin for image and content watermarking with advanced features.

== Description ==

Watermark Manager is a powerful WordPress plugin that allows you to automatically add watermarks to your images. With support for bulk operations, preview functionality, and advanced watermark customization options.

= Features =

* Image watermarking with customizable position, opacity, size, and rotation
* Bulk watermarking capability
* Live preview of watermark settings
* Automatic watermarking of new uploads
* Original image backup and restore functionality
* Advanced settings management
* React-based modern admin interface

= Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* GD or Imagick PHP extension

== Installation ==

1. Upload the `watermark-manager` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Watermark Manager' in your admin menu to configure settings

== Configuration ==

1. **Watermark Image**: Upload or select your watermark image
2. **Position**: Choose where to place the watermark (top-left, top-right, bottom-left, bottom-right, center)
3. **Opacity**: Set watermark transparency (0-100%)
4. **Size**: Adjust watermark size relative to the image (1-100%)
5. **Rotation**: Rotate watermark (0-360 degrees)
6. **Auto-watermark**: Enable/disable automatic watermarking of new uploads
7. **Backup originals**: Enable/disable backup of original images

== Frequently Asked Questions ==

= Can I restore the original image after watermarking? =

Yes, if you have enabled the "Backup originals" option, you can restore the original image from the "Manage Images" section.

= What image formats are supported? =

The plugin supports JPEG, PNG, GIF, and WEBP formats.

= Can I watermark existing images? =

Yes, use the "Bulk Watermark" feature to watermark multiple existing images at once.
